# 基于React+Typescript+Koa搭建的前后台博客系统
> 此项目的本意就是将自己所学的应用上去，并且通过博客记录沉淀下来

## 项目技术栈
- [x] React
- [x] Router
- [x] React-hooks
- [x] Typescript
- [x] Axios
- [x] Scss
- [x] Jest
- [x] Node
- [x] Koa
- [x] MongoDB
- [x] Mongoose
- [x] Docker
- [x] Nginx
- [x] CI & CD
- [x] 七牛云存储
- [ ] ...

## 组件复用
把项目用到的组件（Button、Menu、Model等）都会单独抽离出来，并且使用ReactHooks开发，发布到npm上去，site-next-ui详情[点击](https://github.com/ZAnsder/site-next-ui)
